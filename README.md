# Sustainability Report RAG ChatBot

A web-based Retrieval Augmented Generation (RAG) application that enables users to upload Environmental Sustainability Reports (PDF) from large companies and interactively ask questions about their content.

## Features

- **PDF Upload & Processing** — Upload sustainability reports and automatically extract, chunk, and index the content
- **Intelligent Q&A** — Ask natural language questions about emissions, targets, initiatives, and more
- **Structured Data Extraction** — Download key sustainability metrics as a structured JSON file


## Example Querie
- *"How did the carbon emission reduce between 2024-2025?*
